# Resource Pack

## Introduction

[Cobblemon Trainer Battle Resource Pack](https://github.com/KiwiFlavoredApollo/cobblemon-trainer-battle-resource-pack) can be used as a template.

## Resource Pack Structure

```
resourcepacks/
└── your_resourcepack_name/
    ├── pack.mcmeta
    └── assets/
        └── cobblemontrainerbattle/
            └── textures/
                └── entity/
                    └── trainer/
                        └── slim/
                            ├── leader_brock.png
                            ├── leader_misty.png
                            └── ...

```

## Trainer Textures

- Gym Leaders, Elite Four and Champion textures (skins) can be overridden via resource pack.
