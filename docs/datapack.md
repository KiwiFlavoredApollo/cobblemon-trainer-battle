# Data Pack

## Introduction

[Cobblemon Trainer Battle Data Pack](https://github.com/KiwiFlavoredApollo/cobblemon-trainer-battle-data-pack) can be used as a template.

## Data Pack Structure

```
datapacks/
└── your_datapack_name/
    ├── pack.mcmeta
    └── data/
        └── cobblemontrainerbattle/
            ├── trainer_team/
            │   ├── radicalred/
            │   ├── inclementemerald/
            │   ├── smogon/
            │   └── custom/
            │       ├── custom_trainer_1.json
            │       └── custom_trainer_2.json
            │
            ├── trainer_preset/
            │   ├── radicalred/
            │   ├── inclementemerald/
            │   ├── smogon/
            │   └── custom/
            │       ├── custom_trainer_1.json
            │       └── custom_trainer_2.json
            │
            └── loot_tables/
                └── entities/
                    └── trainer.json
```
